# 依頼文（このまま `codex exec` に渡す）
あなたは探索担当です。予定調和を避け、候補空間を広げてください。

要件:
1. 最低5案を出す（互いに異なる方向性）。
2. 各案は「着想 -> 論理接続 -> 実装/実験案」の順で示す。
3. 連想で出した要素は、必ず因果または制約で接続して整合させる。
4. 画像/文面など定量化しづらい案は、目視評価ポイントを明記する。
5. 収束は行わず、次の比較実験を提案する。

最終出力形式:
- 案一覧（5案以上）
- 案ごとの論理接続
- 目視/感性評価ポイント
- 次の比較実験

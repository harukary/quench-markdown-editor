# テーブル表示サンプル（Quench）

Quench の Live Preview（GFMテーブル装飾）の見た目確認用。

## 1) 基本

| 項目 | 値 |
| --- | --- |
| 名前 | Quench |
| 状態 | OK |
| 数値 | 123 |

## 2) アラインメント

| 左寄せ | 中央 | 右寄せ |
| :--- | :---: | ---: |
| a | b | c |
| 1 | 2 | 3 |

## 3) 長いテキスト（折返し/はみ出し確認）

| Key | Description |
| --- | --- |
| long_text | これはとても長い説明文です。横幅が狭い場合の見え方（折返し/はみ出し/スクロール）を確認します。 |
| url | https://example.com/path/to/something/very/long |

## 4) パイプ（`|`）を含むテキスト

> Markdownの仕様上、セル内の `|` は `\\|` にエスケープする必要があります。

| 入力 | 結果 |
| --- | --- |
| a \\| b | パイプを含む |
| `a | b` | インラインコード内のパイプ |
